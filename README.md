# ECS Fargate Platform (Terraform)

This project provisions a production-style container platform on AWS using Terraform.

## Features
- VPC with public/private subnets
- Application Load Balancer
- ECS Fargate cluster and service
- Security groups and IAM roles
- CloudWatch logging

## Tech Stack
- AWS (ECS, ALB, VPC, IAM, CloudWatch)
- Terraform
- Docker

## Status
Infrastructure deployed and running using nginx container.
